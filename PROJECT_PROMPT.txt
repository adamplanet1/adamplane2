PROJECT: DekoKraft (Static site on GitHub Pages)
STACK: HTML + CSS + Vanilla JavaScript + JSON فقط (بدون Framework, بدون Backend)

GOAL:
موقع ثابت بواجهة Dark/Glass يدعم لغتين AR/DE.
تبديل اللغة لا يغير الروابط ولا يكسر التصميم.
RTL في العربية و LTR في الألمانية.

PAGES:
1) index.html (الصفحة الرئيسية)
- Hero
- قسم "الأقسام" (4 بطاقات رئيسية): Gifts / Decor / Kids / Services
  * يجب أن تكون 2×2 على الهاتف
  * ويجب أن تبقى 2×2 على الكمبيوتر أيضاً (لا تتحول إلى 4×4)
  * بطاقة القسم تحتوي زر "عرض القسم" + يمكن جعل البطاقة نفسها رابط أيضاً.
  * IMPORTANT NAVIGATION:
    زر "عرض القسم" يجب أن يفتح صفحة المنتج (product.html) مباشرة
    عبر أول منتج في هذا القسم (firstProductId لكل قسم) — وليس صفحة category.html

- قسم "منتجات مختارة" (Featured / Random)
  * هذه البطاقات لا تحتوي زر داخلها
  * البطاقة كلها تكون رابط إلى صفحة المنتج product.html?id=...
  * يجب عدم تكرار نفس منتجات الأقسام الرئيسية (استبعاد firstProductId لكل قسم)

- KONTAKT (موحد في كل الصفحات)
  * WhatsApp + Telegram + Email فقط
  * حذف Facebook نهائياً (لا كود ولا SVG)
  * أسفل KONTAKT يظهر Logo كبير داخل مربع الزجاج

2) category.html (اختياري)
- شبكة منتجات قسم معين حسب ?cat=
- كل بطاقة تفتح product.html?id=...
- KONTAKT
(ممكن إبقاؤها للعرض، لكن الزر الرئيسي في الصفحة الرئيسية لا يذهب إليها)

3) product.html (صفحة المنتج)
- عرض المنتج بشكل مُكبر:
  * صورة كبيرة رئيسية
  * عنوان + وصف قصير جميل تحت/بجانب الصورة
- أسفل/جانب العرض:
  * قائمة عمودية Thumbnails لمنتجات نفس القسم (صغير)
  * عند الضغط على Thumbnail:
    - تتغير الصورة الكبيرة + العنوان + الوصف إلى المنتج المختار
    - ويتغير رابط الصفحة (history.replaceState) إلى id الجديد بدون إعادة تحميل
- KONTAKT في الأسفل

CRITICAL IMAGE RULE:
صور المنتجات ليست gift-001.webp
بل:
gift-001-600.webp و gift-001-1200.webp (قد يوجد واحد منهما)
في products.json نستخدم:
imageBase: "assets/images/products/gifts/gift-001"
والكود يختار:
- -1200.webp إذا موجود
- وإلا -600.webp إذا موجود
- وإلا Placeholder بدون كسر التصميم

VISUAL DIFFERENTIATION:
- Cards الأقسام الرئيسية: زجاج نظيف بدون “بخار أزرق”
- Cards المنتجات المختارة: يوضع Overlay ديكور (Blue Fog / Glass Vapor) فوق الصورة فقط أو فوق الكارت
  (بدون أن يبدو كصورة مكسورة، وبشكل واضح أنه ديكور)

WORKFLOW:
عند طلب تعديل ملف: اطبع الكود كامل لذلك الملف فقط.
(الآن طباعة كاملة للمشروع مطلوبة.)
