# DekoKraft (Static)

- HTML + CSS + Vanilla JS + JSON
- GitHub Pages friendly

## Pages
- `index.html` Home
- `category.html?cat=gifts|decor|kids|services`
- `product.html?id=gift-001` etc.

## Images rule
Use `imageBase` in `assets/data/products.json`
Example:
`assets/images/products/gifts/gift-001`
App resolves:
- `...-1200.webp` then `...-600.webp`
